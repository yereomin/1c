Справочник предназначен для демонстрации работы подсистем Анкетирование, Взаимодействия, Даты запрета изменения, Контактная информация, Пользователи, Префиксация, Присоединяемые файлы, Работа с файлами, Свойства, Управление доступом.

В подсистеме Анкетирование элементы справочника используются в качестве респондентов, имеющих доступ к базе как внешние пользователи.

В подсистеме Взаимодействия элементы справочника используются в качестве контактов взаимодействий.

В подсистеме Даты запрета изменения элементы справочника могут запрещаться для изменения датой в разрезе раздела Торговля с объектами Демо: Группы доступа партнеров.

В подсистеме Контактная информация элементы справочника являются объектами-владельцами контактной информации.

В подсистеме Пользователи элементы справочника используются в качестве объекта авторизации внешних пользователей.

В подсистеме Префиксация объектов к коду элементов справочника добавляется префикс информационной базы.

В подсистеме Присоединяемые файлы элементы справочника используются в качестве объектов-владельцев файлов вложений.

В подсистеме Работа с файлами элементы справочника используются в качестве объектов-владельцев файлов вложений.

В подсистеме Свойства элементы справочника являются объектами-владельцами дополнительных реквизитов и сведений. При этом набор дополнительных свойств для конкретного элемента справочника зависит от значений реквизитов Клиент, Поставщик, Конкурент и Прочие отношения. В списке показан пример вывода дополнительных реквизитов для произвольных объектов.

В подсистеме Управление доступом элементы справочника являются значениями доступа. Значения доступа Демо: Партнеры косвенно элементами Демо: Группы доступа партнеров назначаются в группах доступа в списке разрешенных значений по виду доступа Демо: Группы партнеров.
